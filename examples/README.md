# SimLab examples

This directory contains small, copyable examples for SimLab features. It is not limited to MCP.

## Included examples

| Path | Purpose |
|------|---------|
| [custom-cli-tools/](custom-cli-tools/) | Example coding environment with custom CLIs, mounted fixtures, a local task, and a verifier. |
| [mcp-servers-url-only.json](mcp-servers-url-only.json) | Minimal URL-based MCP config. |
| [mcp-servers-command-only.json](mcp-servers-command-only.json) | Minimal command-based MCP config with an API-key env var. |
| [mcp-servers-mixed.json](mcp-servers-mixed.json) | Minimal mixed MCP config. |
| [mcp-servers-local-demo.json](mcp-servers-local-demo.json) | URL config for the local demo MCP server. |
| [demo-mcp-server/](demo-mcp-server/) | Tiny local MCP server for smoke tests and development. |
| [tasks/list-tools.json](tasks/list-tools.json) | Small local task for verifying tool discovery. |

For MCP setup, env var handling, and end-to-end usage, see [QUICKSTART](../QUICKSTART.md).
